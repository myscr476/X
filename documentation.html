📘 X# Interpreter Documentation

Welcome to X# Interpreter v2.5.1, a shell-based command interface and scripting environment created by Athaya.

🧰 What is X#?

X# is a custom scripting language that mimics Bash + Python with simplified syntax. It also has its own package system, virtual shell, and CLI-based OS interface.

💻 Getting Started

When you start main.py, you'll enter the X# shell:

~ $

Type help to see all commands.

📦 Built-in Commands

  • mkdir [name] – Create a folder
  • dir – List directories
  • cd [path] – Change directory
  • pwd – Print current directory
  • tree – Show file tree
  • math [expr] – Calculator
  • pkg install [pkg] – Install package (internal & pip)
  • pyrun – Run Python interpreter
  • alias [a] [b] – Create alias
  • xeditor – Run X# code editor
  • windai – Open WindAI assistant
  • selfdestruct – Self-destruct (exit program)

🧠 WindAI

WindAI is an experimental chatbot assistant that responds to simple queries when installed using pkg install windai.

✍️ X# Syntax Sample

nugget("Hello World")
